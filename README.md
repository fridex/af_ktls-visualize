## Benchmarks

*You can show benchmarks with:*

```sh
sudo ./client                   \
    --sendfile-mtu 1400         \
    --sendfile-buf file.bin     \
    --drop-caches               \
    --server-host localhost     \
    --server-port 5557          \
    --sendfile file.bin         \
    --payload 1371              \
    --json                      \
    --tls                       \
| ./visualize.py                \
    --html-browse               \
    --html-stats                \
    --output-dir /tmp 
```

<br>

`./client`  *is from the **AF_TLS** socket tool.*

<br>

*See*  `--help`  *for more information.*
